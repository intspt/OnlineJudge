online judge by spt

=================================================================================

环境要求:
1. linux

2. 安装python的相应模块

    第三方: (flask, flask-login, flask-sqlalchemy, flask-wtf, sqlalchemy)

    其他:   lorun(https://github.com/lodevil/Lo-runner)

    
3. 安装mysql

=================================================================================

运行:

首先配置数据库URI参数并建立对应数据库(utf-8)

然后运行init.py进行初始化

最后分别运行run.py与judging.py
